# Week 5,6 - MERN Stack

This folder contains Week 6 learning and implementation in the MERN Stack journey.

## Topics Covered

- Backend Development with Node.js
- Express.js Basics
- REST API Development
- Middleware
- MongoDB Integration
- CRUD Operations

## Technologies Used

- Node.js
- Express.js
- MongoDB
- JavaScript 

## How to Run

1. Clone the repository:
```bash
git clone https://github.com/KoshikaJyothi/MERN.git
```

2. Navigate to the folder:
```bash
cd MERN/WEEK-6
```

3. Install dependencies:
```bash
npm install
```

4. Start the server:
```bash
npm start
```

## Purpose

To build backend applications and understand API development using the MERN stack.
